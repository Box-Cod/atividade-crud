# atividade-crud

## End-points

### Personagem
- (GET) http://localhost:8081/personagens -> findAll()
- (GET) http://localhost:8081/personagens/1 -> findById(id)
- (PUT) http://localhost:8081/personagens/1 -> update(id)
- (DELETE) http://localhost:8081/personagens/1 -> delete(id)

### Item

- (GET) http://localhost:8081/itens -> findAll()
- (GET) http://localhost:8081/itens/1 -> findById(id)
- (PUT) http://localhost:8081/itens/1 -> update(id)
- (DELETE) http://localhost:8081/itens/1 -> delete(id)
